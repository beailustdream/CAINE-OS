# 🎭 CAINE OS

A cinematic operating system inspired by **The Amazing Digital Circus**.

CAINE OS is an interactive sci-fi/horror styled interface built with React + Vite, featuring:

* 📼 VHS / CRT effects
* 👁 Surveillance system
* 🎪 Performer monitoring
* 🩸 Corruption mode
* 💻 Interactive terminal
* 🔊 System sounds
* ✨ Animated UI

# 📸 Features

## 👁 Surveillance System

Monitor Digital Circus performers through cinematic CCTV interfaces.

## 💻 Interactive Terminal

Execute commands such as:

* `help`
* `corrupt`
* `restore`
* `performers`

## 📼 VHS Effects

Retro CRT overlays, scanlines and glitch effects.

## 🎭 Performer Database

Profiles and monitoring system for circus performers.

## 🩸 Corruption Mode

Reality instability effects with visual glitches and audio distortion.
